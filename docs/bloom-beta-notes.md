# Bloom Beta Notes

Bloom is currently being prepared for a small public beta feedback round.

## Current Beta Focus

- Public Overview page clarity
- Demo mode entry
- Login/create account modal flow
- Accessibility controls
- Mobile layout
- Moments page polish
- Feedback collection flow

## Current Beta Limitations

Bloom is still in active development. Some features are currently frontend-only or demo-based.

Current limitations:

- Feedback is not yet connected to a backend database
- Account creation and authentication are still being stabilised
- Some user data is still stored locally
- Demo mode uses sample data
- Backend persistence, production authentication, and saved user preferences are planned for later versions

## Tester Incentive Idea

For the first beta round, the first 10 helpful testers may receive free lifetime access to Bloom's future paid personal features, if paid plans are added later.

Suggested wording:

> As a thank-you, the first 10 people who give useful beta feedback will receive free lifetime access to Bloom’s future paid personal features, if Bloom adds paid plans later.
