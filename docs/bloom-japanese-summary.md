# Bloom Japanese Summary / 日本語サマリー

## 概要

Bloomは、落ち着いたルーティン管理とタスク進行を支援するアクセシビリティ重視のWebアプリです。

ニューロダイバージェントフレンドリーな体験を重視し、ユーザーがステップごとのルーティン、集中タスク、進捗、振り返りを分かりやすく管理できるように設計しています。
