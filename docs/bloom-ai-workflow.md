# Bloom AI-Assisted Development Workflow

Bloom uses supervised AI-assisted development as part of the engineering workflow.

## AI Tools Used

- Claude Code
- Codex
- ChatGPT

## Use Cases

AI tools were used for:

- Code review support
- Git diff review
- Security and configuration checks
- Debugging guidance
- README and documentation drafting
- Refactoring suggestions
- Frontend UI polish planning
- Technical recap generation

## Developer Control

AI-generated suggestions were not committed automatically.

The developer manually reviewed diffs, tested changes, controlled Git commits, and made final implementation decisions.

## Summary

Used Claude Code and ChatGPT as supervised AI pair-programming tools for code review, debugging support, backend security/configuration checks, and documentation, while manually reviewing diffs, testing changes, and managing Git commits.
